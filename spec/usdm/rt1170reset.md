# 【rt1170reset】RT1170-EVK USBリセットツール

## 要求

Seeeduino XIAO RP2040 を USB 接続し、PIN7 で TLP241A を駆動して RT1170-EVK の J1 20 ピンの 15 番と 16 番を短時間導通させることでボードをリセットするファームウェアを提供する

### 理由

RT1170-EVK のリセット操作を手元のシェルやコーディングエージェントから安全かつ再現可能に自動実行できるようにするため

### 説明

現行実装は USB CDC シリアル経由の ASCII コマンドで PIN7 のオンオフとパルス幅を制御するファームウェアであり、専用ホスト CLI は未実装である
