# 【rt1170reset_R01.DM02】USB CDCデバイス識別

## 要求

ホストが対象デバイスを自動検出できるように USB CDC デバイスとして一意な識別情報を公開する

### 理由

複数のシリアルデバイスが接続された環境でも CLI が誤った機器に接続しないようにするため

### 説明

USB シリアルの通信内容は ASCII コマンドだが、デバイス選択は USB の識別子で行う

## 仕様

| 種別 | 内容 | 仕様番号 |
| --- | --- | --- |
| 仕様 | USB クラスは CDC ACM 相当のシリアルデバイスとして列挙される | 【rt1170reset_R01.DM02-01】 |
| 仕様 | USB VID/PID は `0x1209:0x0001` を使用する | 【rt1170reset_R01.DM02-02】 |
| 仕様 | Product 文字列は `EVKB USB Reset` を公開する | 【rt1170reset_R01.DM02-03】 |
| 仕様 | Serial Number 文字列はホストが選択キーとして利用できる形式で公開する | 【rt1170reset_R01.DM02-04】 |
