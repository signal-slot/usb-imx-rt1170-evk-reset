# 【rt1170reset_R01】USB CDCリセットインタフェース

## 要求

USB CDC シリアル経由でホストからのリセット要求を受け付け、単純で機械処理しやすいテキストプロトコルで応答する

### 理由

ホスト CLI やコーディングエージェントが余計な対話なしで RT1170-EVK を制御できるようにするため

### 説明

コマンドは行区切り ASCII とし、ファームウェアはホストからのコマンド単位で結果を返す
