# 【rt1170reset_R02.DM02】パルス幅と安全復帰

## 要求

リセット出力のパルス幅を制御し、起動時・異常時を含めて必ず非アサート状態へ戻す

### 理由

EVK のリセット信号を保持し続ける故障モードを避けるため

### 説明

コマンド処理中はブロッキングでもよいが、パルス中も USB の列挙と I/O が破綻しないこと、およびパルス完了後の Low 復帰は必須とする

## 仕様

| 種別 | 内容 | 仕様番号 |
| --- | --- | --- |
| 仕様 | `RESET` のパルス幅未指定時は 100ms を使用する | 【rt1170reset_R02.DM02-01】 |
| 仕様 | 指定可能なパルス幅は 10ms 以上 1000ms 以下とし、範囲外はその範囲に丸める | 【rt1170reset_R02.DM02-02】 |
| 仕様 | 電源投入後は USB 列挙前にリセット出力を Low に設定する | 【rt1170reset_R02.DM02-03】 |
| 仕様 | パルス中は USB デバイスを poll し続け、受信した CDC データは処理せず破棄する | 【rt1170reset_R02.DM02-04】 |
| 仕様 | パルス終了後は必ずリセット出力を Low に戻し、入力行バッファを空に戻す | 【rt1170reset_R02.DM02-05】 |
| 仕様 | 長すぎる入力行や不正入力ではリセット出力をアサートしない | 【rt1170reset_R02.DM02-06】 |
