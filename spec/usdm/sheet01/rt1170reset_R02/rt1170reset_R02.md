# 【rt1170reset_R02】GPIOパルス制御

## 要求

XIAO RP2040 の PIN7 を用いて TLP241A をオンオフし、RT1170-EVK のリセット信号に対して安全なワンショットパルスを印加する

### 理由

ホストが `RESET` コマンドを送るだけで EVK のリセット動作を再現できるようにするため

### 説明

XIAO の PIN7 は BSP 上では `GPIO1` を出力として扱い、High でフォトリレーを導通させる
