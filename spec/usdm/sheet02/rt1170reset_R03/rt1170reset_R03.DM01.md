# 【rt1170reset_R03.DM01】サブコマンド提供

## 要求

CLI からファームウェアの主要機能を直接呼び出せるようにサブコマンドを提供する

### 理由

利用者が USB CDC の生プロトコルを覚えなくても必要な操作を実行できるようにするため

### 説明

ここでの CLI は将来実装対象であり、最終的に USB CDC へ ASCII コマンドを送るフロントエンドとして振る舞う

## 仕様

| 種別 | 内容 | 仕様番号 |
| --- | --- | --- |
| 仕様 | `ping` サブコマンドはデバイスへ `PING` を送り、`PONG` を確認する | 【rt1170reset_R03.DM01-01】 |
| 仕様 | `off` サブコマンドはデバイスへ `OFF` を送り、リセット出力解除を要求する | 【rt1170reset_R03.DM01-02】 |
| 仕様 | `reset` サブコマンドはデバイスへ `RESET` または `RESET <ms>` を送り、パルス実行を要求する | 【rt1170reset_R03.DM01-03】 |
| 仕様 | `reset` サブコマンドはパルス幅オプション未指定時に 100ms を既定値として扱う | 【rt1170reset_R03.DM01-04】 |
| 仕様 | `bootsel` サブコマンドはデバイスへ `BOOTSEL` を送り、ファームウェア更新用の USB ブートモード遷移を要求する | 【rt1170reset_R03.DM01-05】 |
