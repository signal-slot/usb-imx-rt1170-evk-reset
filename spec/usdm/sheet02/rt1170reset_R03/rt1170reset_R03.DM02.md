# 【rt1170reset_R03.DM02】安定した入出力と終了コード

## 要求

将来実装する CLI の標準入出力と終了コードを安定化し、スクリプトやエージェントから判定可能にする

### 理由

人手と自動化の両方で同じコマンドを安全に利用するため

### 説明

成功判定はデバイス応答と接続状態に基づいて行う

## 仕様

| 種別 | 内容 | 仕様番号 |
| --- | --- | --- |
| 仕様 | CLI は対話プロンプトを表示しない | 【rt1170reset_R03.DM02-01】 |
| 仕様 | 成功時は終了コード 0 を返す | 【rt1170reset_R03.DM02-02】 |
| 仕様 | タイムアウト、デバイス未検出、複数候補、期待外の応答、通信失敗では 0 以外の終了コードを返す | 【rt1170reset_R03.DM02-03】 |
| 仕様 | 成功時の結果は 1 行で標準出力へ表示し、失敗理由は標準エラー出力へ表示する | 【rt1170reset_R03.DM02-04】 |
| 仕様 | `--help` で使用方法とオプション一覧を表示できる | 【rt1170reset_R03.DM02-05】 |
