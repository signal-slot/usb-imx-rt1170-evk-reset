# 【rt1170reset_R04.DM01】デバイス発見

## 要求

将来の CLI は既定動作として接続済み USB デバイスから対象ファームウェアを探索する

### 理由

利用者が毎回シリアルポート名を調べなくてもコマンドを実行できるようにするため

### 説明

探索条件はファームウェアが公開する USB 識別情報を用いる

## 仕様

| 種別 | 内容 | 仕様番号 |
| --- | --- | --- |
| 仕様 | 自動検出時は USB VID/PID `0x1209:0x0001` を優先条件とする | 【rt1170reset_R04.DM01-01】 |
| 仕様 | Product 文字列 `EVKB USB Reset` を照合条件として利用できる | 【rt1170reset_R04.DM01-02】 |
| 仕様 | 候補が 0 台のときは未検出として失敗終了する | 【rt1170reset_R04.DM01-03】 |
| 仕様 | 候補が複数台のときは曖昧として失敗終了し、明示指定方法を案内する | 【rt1170reset_R04.DM01-04】 |
