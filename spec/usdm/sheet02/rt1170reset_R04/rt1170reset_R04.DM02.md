# 【rt1170reset_R04.DM02】明示選択と接続制御

## 要求

将来の CLI は利用者が接続先を明示でき、通信失敗時は有限時間で終了する

### 理由

複数台接続環境や不安定な USB 環境でも運用不能にならないようにするため

### 説明

明示指定のキーは OS 依存のポート名か USB シリアル番号を想定する

## 仕様

| 種別 | 内容 | 仕様番号 |
| --- | --- | --- |
| 仕様 | CLI は `--port` で OS のシリアルポート名を明示指定できる | 【rt1170reset_R04.DM02-01】 |
| 仕様 | CLI は `--serial-number` で USB シリアル番号を明示指定できる | 【rt1170reset_R04.DM02-02】 |
| 仕様 | CLI は応答待ちの上限時間を指定するオプションを持つ | 【rt1170reset_R04.DM02-03】 |
| 仕様 | タイムアウト時は処理を中断し、失敗終了する | 【rt1170reset_R04.DM02-04】 |
