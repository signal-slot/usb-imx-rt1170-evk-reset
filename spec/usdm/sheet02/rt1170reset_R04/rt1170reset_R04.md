# 【rt1170reset_R04】対象デバイス選択要求

## 要求

将来のホスト側 CLI は接続先の XIAO RP2040 を識別し、対象デバイスが一意に決まる場合だけコマンドを送信する

### 理由

他の USB シリアル機器へ誤って `RESET` を送る事故を防ぐため

### 説明

これは将来要求であり、現行実装では CDC デバイス選択をホスト利用者側が担う
