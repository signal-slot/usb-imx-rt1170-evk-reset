# 【rt1170reset_R05】ハードウェア接続条件

## 要求

XIAO RP2040、TLP241A、RT1170-EVK の接続関係を固定し、リセット経路を明示する

### 理由

ソフトウェア仕様と現物配線の対応を崩さずに保守できるようにするため

### 説明

USB は XIAO RP2040 の電源供給と CDC 通信に使用し、RT1170-EVK 側のリセット線とは TLP241A で絶縁する
