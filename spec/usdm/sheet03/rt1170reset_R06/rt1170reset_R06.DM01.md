# 【rt1170reset_R06.DM01】利用手順と前提の明文化

## 要求

リポジトリ内の文書から、必要な配線、ファームウェア書き込み、ビルド設定、CLI 実行方法を読み取れるようにする

### 理由

エージェントや新規利用者が外部の口頭説明なしにセットアップできるようにするため

### 説明

手順はホスト OS 差異をまたいで読み替え可能な粒度で記述する
